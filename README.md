# School Management System

## Overview

School Management System is a Python-based console application designed to manage student records, teacher information, class timetables, and academic reports. The project uses CSV files for data storage and Pandas for data management.

## Features

* Admin Login System
* User Login System
* Student Record Management
* Teacher Record Management
* Class Timetable Viewer
* Student Information Portal
* Subject Marks Management
* Academic Report Generation
* Marks Visualization using Matplotlib

## Technologies Used

* Python
* Pandas
* Matplotlib
* CSV File Storage

## Installation

1. Clone the repository

```bash
git clone https://github.com/Bhautikcodi/School-Management-System-Python.git
```

2. Install dependencies

```bash
pip install pandas matplotlib
```

3. Run the project

```bash
python main.py
```
